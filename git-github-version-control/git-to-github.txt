#########################################################################################

steps to setup git on local system and do ssh setup between local system and github account

#########################################################################################



#setup ssh authentication between local git repo and github remote repo

ssh-keygen -t rsa -b 4096 -C "any comment"

 #Add the SSH key to the SSH agent:

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa

git init
git add .
git commit -m "commit message"

git config --global user.name "Your Name" #use github account username, having same make its easy
git config --global user.email "your.email@example.com" #use github noreply email here

git remote -v # will list if any remote github repo linked to local git repo

git remote add origin master #branch and repo name can be as per your specifications

git push -u origin master 

#####################################################################################################

steps to create new github repo and add it as remote repo to your local git repo

#####################################################################################################

Create a New GitHub Repository:
Go to the GitHub website (https://github.com/) and log in to your account.
Click on the "+" sign in the top right corner of the page and select "New repository."
Fill in the repository name, add a description (optional), choose public or private visibility, and initialize with a README file if you want.
Click on the "Create repository" button.

Copy the Repository SSH URL:
On the GitHub repository page, copy the SSH URL. It will look like git@github.com:username/repo.git.

Navigate to Your Local Git Repository:
Open a terminal or command prompt.
Navigate to the directory of your existing local Git repository using the cd command.

Add GitHub Repository as a Remote:
Use the following command to add the GitHub repository as a remote. Replace the URL with the one you copied in step 2.
#git remote add origin git@github.com:username/repo.git

Push to GitHub:
Now, you can push your local repository to GitHub using the following commands:
#git push -u origin master
This assumes you're pushing the master branch. If you're on a different branch, replace master with your branch name.
If you've set up your SSH keys correctly, you shouldn't be prompted for a username or password.

Verify on GitHub:
Refresh your GitHub repository page, and you should see your code and files there.
