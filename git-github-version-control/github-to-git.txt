this is to get the code on your local git repo from github repo

##############################################################################################################

 first install git on cloud instance or local system you are working on ;
then do git global config of username and email, email should be no-reply email provided by github and username your github username ;
have to do ssh authentication setup between github remote repo and git local repo to be able to do push and pull between remote and local repo

sudo yum install git -y

#setup ssh authentication between local git repo and github remote repo

ssh-keygen -t rsa -b 4096 -C "any comment"

 #Add the SSH key to the SSH agent:

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa



git config --global user.name "Your Name" #use github account username, having same make its easy
git config --global user.email "your.email@example.com" #use github noreply email here

# then add this github remote repo and do pull command ;
# go to directory where you want to get code on local systen and do clone

git clone "git@github.com:username/repo-name.git" #this is ssh url of github repo; it  will not ask for username and passwd as ssh already done initially
